Text Paragraphs

HTML is based on tags. You can tell the web browser that you want to include a paragraph of text in your web page by enclosing the text in the paragraph tags.

Opening and Closing Tags

In the image above, we can see the opening and closing tags.
<p> is the opening paragraph tag. It defines the start of the element.
</p> is the closing tag, which is used to define the end of the element with the foward slash.

Many HTML elements require both opening and closing tags, also called container tags.
